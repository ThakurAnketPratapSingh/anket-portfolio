### 📲 Just For You (J4U)
> USSD-based real-time offer interface

A USSD customer interface enabling telecom subscribers to browse and select personalised offers generated by MRTM in real time — built for ultra-low-latency, multilingual rendering, and high-concurrency session management.

**Impact:**
- ⬆️ 25% improvement in API response time
- ✅ Real-time offer retrieval via gRPC + Kafka — sub-200ms end-to-end
- ✅ Distributed session management via Hazelcast for concurrent USSD flows

**Architecture:**
```
Spring Boot → gRPC (MRTM offers) → Kafka (async logging) → Hazelcast (session) → USSD Gateway (<200ms)
```

**Stack:** `gRPC` `Kafka` `Spring Boot` `Hazelcast` `USSD` `JPA/Hibernate`

---

### 🔀 Integration Gateway (CUIG)
> Centralised upstream/downstream integration hub

A centralised API gateway handling all upstream and downstream system communication — supporting both synchronous HTTP and asynchronous Kafka flows with XSLT-based message transformation. Configuration-driven: new integrations require zero code deployments.

**Impact:**
- ⬇️ 90% reduction in deployment effort — new integrations via config only
- ✅ Dual-protocol: HTTP (sync) + Kafka (async) in a single gateway
- ✅ XSLT transformation engine for format-agnostic message processing (XML ↔ JSON)

**Modules:**
```
GUI · Config Service · Gateway Service · User Service · Event Processor · Interactive · Fulfilment
```

**Stack:** `Spring Boot` `Apache Kafka` `Consul` `PostgreSQL` `XSLT` `REST APIs` `Microservices`
